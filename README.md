## How to install the notes package

In your projects `package.json` dependencies:

```
"<your-choice-of-name>": "https://gitpkg.now.sh/nbdehlen/notes-app/dist?master"
```

## How to run the demo

`cd demos && npm i` in the terminal and start a live server
